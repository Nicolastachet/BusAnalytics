Description of the files in this folder

1 - bank.csv, bank-full.csv, bank-full2.csv are the three dataset we used for this project

2 - Decision_tree.R is the R file to fit the decision tree models on the dataset

3 - Logistic_Regression.R is the R file to fit the logistic regression on the dataset

4 - PCAKNN.R is the R file to fit the kNN models on the dataset and to compute the PCA analysis

5 - ROC.R is the R file to compute the ROC of the three previous models

6 - server.R, ui.R, www folder are the files for the Shiny. Just open either server.R or ui.R with RStudio and click "Run App"
